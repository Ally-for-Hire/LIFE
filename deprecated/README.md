# Deprecated

The original vanilla-JS / HTML / Node prototype of LIFE. It has been fully
replaced by the native Rust (egui) application in [`../life-rs/`](../life-rs/).

Kept here for reference only — nothing in this folder is built or run. Safe to
delete once you're confident you won't need it.

- `js/` — the old simulation classes (entity, clan, world, brain, trainer, …).
- `server.js` / `training-worker.js` — the old Node runner and worker-thread trainer.
- `index.html` / `fast.html` — the old browser UIs.
- `test/smoke.js` — the old headless JS smoke test.
